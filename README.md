🛠️ Especificaciones Técnicas del ADN Nigeriano
A diferencia de los registros anteriores, este contrato clasifica los platos por su comportamiento físico y su base lipídica:

Tipo de Espesante (string): Identifica el agente que define la textura (Ej: Ogbono para elasticidad, Egusi para grumos proteicos).

Uso de Aceite de Palma (bool): Validador de identidad visual y perfil de sabor graso ancestral de la región.

Escala de Picante (uint256): Medición de la intensidad del Scotch Bonnet (Ata Rodo), validada de 0 a 10.

Sistema de Feedback: Funciones de darLike y darDislike para la interacción comunitaria.
